Trojan Image Processor

What The Projects About

This project is a web-based image processor that allows users to upload multiple images and apply grayscale conversion, resizing, and blurring effects easily. Using multithreading on the backend, images are processed efficiently and returned in a single downloadable ZIP file.

Features

  • * Batch image upload
  • * Grayscale conversion
  • * Custom resizing
  • * Adjustable blur effects
  • * Fast multithreaded processing
  • * Download all processed images in a ZIP archive
